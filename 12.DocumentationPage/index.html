HTML Basics

HTML stands for HyperText Markup Language...

It is the standard language for creating web pages...

<html></html> <body></body>
  • HTML elements are represented by tags
  • HTML can be embedded with CSS and JavaScript
CSS Fundamentals

CSS stands for Cascading Style Sheets...

It describes how HTML elements are to be displayed on screen...

body { color: black; } h1 { font-size: 2em; }
  • CSS controls the layout of multiple web pages
  • External stylesheets can be used to define styles
Python Overview

Python is an interpreted, high-level programming language...

It emphasizes code readability with its notable use of significant whitespace...

print('Hello, World!') def my_function():
  • Python supports multiple programming paradigms
  • It has a large standard library
SQL Guide

SQL stands for Structured Query Language...

It is used to communicate with databases...

SELECT * FROM users; INSERT INTO users (name, age) VALUES ('Alice', 30);
  • SQL can execute queries against a database
  • SQL can retrieve data from a database
